Le plugin «Simple Photos Contest» propose de gérer un concours de photos en toute simplicité.

== Description ==

= Concernant les photos =

* Permet de téléverser des photos, leur ajouter une description et le nom du photographe.
* Les photos peuvent être triée à la souris.
* Elles ne sont pas ajoutées au gestionnaire de média de Wordpress mais résident dans leur propre répertoire afin de ne pas encombrer la gestion du site.
* 2 images sont calculées par le plugin: une miniature et une image pour visualisation. L'originale de la photo est conservée mais pas affichée.

= Concernant les votes =

* Le vote est ouvert et fermé automatiquement selon les dates configurées
* Plusieurs fréquences sont proposées pour les votes: un seul vote pour une seule photo, un vote toutes les x heures.
* Un export des votes au format tableur permet à tout moment de calculer des statistiques.
* Pour éviter les abus le vote est possible après s'être authentifié soit par email, soit avec son compte Google ou Facebook.

== Installation ==

1. Téléverser le répertoire `simple-photos-contest` dans le répertoire `/wp-content/plugins/`
1. Activer le plugin via l'entrée 'Plugins' du menu WordPress
1. Placer le code `[SimplePhotosContest]` dans votre page
