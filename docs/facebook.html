1.1.2 Identification des votants

Les votants peuvent s'identifier avec leur compte Facebook. Pour permettre cette fonction il faut déclarer le site auprès de Facebook en créant une application. Les identifiants associés à cette application seront utilisés pour la configuration du plugin.

Création d'une application Facebook

  1. Se connecter à Facebook, de préférence avec un compte en rapport avec le site internet sur lequel est installé le plugin
  2. Utiliser le menu "créer une application" (voir écran1, écran2)
  3. Renseigner le "Nom de l'application", le champs "Espace de nom de l'application" est facultatif, malgré l'éventuel message rouge "Échec" (voir écran3).
  4. Sur la page de configuration de l'application, ouvrir la section "Site web avec une authentification Facebook" est renseigner l'"Url du site" (voir écran4)
  5. En haut de la page de configuration de l'application il y a les deux valeurs "App ID" et "App Secret" qu'il va falloir utiliser pour configurer le plugin.

Configuration du plugin

  1. Sur la page de configuration du plugin, dans la section "Options avancées" recopier
  2. la valeur "App ID" dans le champs "Facebook client id"
  3. la valeur "App Secret" dans le champs "Facebook secret key"
  4. Enregistrer les modifications, et l'identification par Facebook doit être opérationnelle
